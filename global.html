getOrDefault(o, path, def) → {any}

Helper method that actually creates the target if it does not exist using the default value. Object-path.get with default option does not sets the default value on the target path.
Parameters:
Name Type Description
o Object The object containing the path
path String | Array a path to the target property inside the object
def any The default value to be used if the target does not exist. This value will be saved at the provided path inside the provided object
